Informational Guide

Sections
  • Report Card
    • The Report Card will direct the user to the Report page
  • Directory Card
    • The Directory Card will direct the user to the Directory page
  • Add User Card
    • The Add User Card will direct the user to the Add Employee page
  • Remove User Card
    • The Remove User Card will direct the user to the Remove Employee page
  • Informational Guide Card
    • The Informational Guide Card will direct the user to the Informational Guide page
  • Settings Card
    • The Settings Card will direct the user to the Settings page
  • Search by Name
    • The Search by Name dropdown is used to search for current employees
      • Employee searches can be conducted by selecting the employee's name from the dropdown menu
      • Click "Get Info" after an employee name is selected to display Employee Information. 
  • Employee Information Card
    • The Employee Information Card displays the information for the employee that has been selected via the dropdown menu
    • The Employee Information Card displays the following employee details, including:
      • Employment
      • Department
      • Position
      • Title
  • Employee Search
    • Allows administrator to look up individual employees by department, title, name or Employee ID
      • A dropdown menu will sort employees based on their department, title, last name, first name or Employee ID
      • Once a selection has been made, click the "Select" button to sort the table
      • The table will populate below with a list of sorted employees
  • Add an Employee to the system
    • An empty form will be generated with various text fields
    • Some of the text fields will be required, such as personal information, position, title and department, while others may be left empty
    • The form will not submit if a required field is left empty
    • Once all fields have been completed, click "Submit" button
    • The employee will now be added to the database and can be located throughout the application
  • Remove an Employee from the system
    • Select the employee's name that should be removed from the dropdown list
    • Click "Select" 
    • Selection must be confirmed before employee will be removed
    • Note: The selected employee will remain in the database for archival purposes, but will be marked as "inactive" during this process
  • Print Button
    • The print button can be clicked from any page in EMS in order to print a copy of the current page
    • A pop-up window will appear in order to configure print settings
  • Log User Out
    • When "Logout" is clicked, the current user will be logged out
